Data Management Dashboard

Welcome to the Admin Dashboard! Here, you'll find easy access to User, Supplier, Stock, Incoming Goods, Outgoing Goods, and Borrow Goods. The Log In button is at the top right. In the center, there's a welcoming "Admin Dashboard" title with a "Welcome to Data Management" message. This dashboard helps you manage data efficiently with Laravel, Tailwind CSS, and Vue.js.

DOCCURE

This website allows users to search for doctors, clinics, and hospitals, as well as book appointments. The main section features a search bar that simplifies finding services based on location and the type of medical care needed. The page also highlights clinic specializations and provides quick access to login or sign-up options. The design is clean, modern, and emphasizes ease of navigation.

Personal Financial Record

The image above showcases a React Native-based financial application for tracking income and expenses. The interface displays the total balance, a financial trend graph, and a transaction list with options to add and edit entries. The design is simple, responsive, and user-friendly.

To-Do App

The image above shows the interface of a mobile application built with React Native, featuring a task list. The app has a header that reads "Task Notes," an input field to add new tasks with a plus (+) icon, and a list of tasks displayed in cards containing the task name and a trash bin icon to delete them. The app is designed with a minimalist and responsive style, ideal for noting and managing daily tasks.

AbsenKita

The image above shows the interface of the "AbsenKita" app, highlighting features such as online attendance, attendance from home, leave requests, as well as the login screen, email verification, and more. Created with Figma, the design uses a consistent orange color scheme, simple illustrations, and a user-friendly layout.

Mountain Adventure Guide

The image above is a design of a landing page with a mountain climbing theme, created with Figma. The design features captivating nature visuals, an inspiring main headline, and segmented content that includes a climbing guide with a combination of text and images. The appearance is modern and professional, ideal for attracting nature enthusiasts.
